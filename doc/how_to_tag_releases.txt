Run this command after changing the version information:

svn copy https://code.kirix.com/svn/strata/trunk https://code.kirix.com/svn/strata/tags/app_4_5_3 -m "tagged version 4.5.3"


To delete an svn:externals property, run
svn propdel svn:externals


To get a raw version of the tree without .svn directories
svn export https://code.kirix.com/svn/strata/tags/app_4_5_3 strata
