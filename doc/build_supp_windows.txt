Zlib 64-bit (not yet used)
-----------

1) nmake /f win32\Makefile.msc


Openssl 32-bit
--------------

1) Download Activestate Perl
2) Open Visual Studio Command Prompt and go to the openssl dir
3) perl Configure debug-VC-WIN32 --prefix=%CD% no-asm
4) ms\do_ms
5) nmake -f ms\nt.mak all
6) perl Configure VC-WIN32 --prefix=%CD% no-asm
7) ms\do_ms
8) nmake -f ms\nt.mak all


Openssl 64-bit
--------------

1) Download Activestate Perl
2) Open Visual Studio Command Prompt and go to the openssl dir
3) perl Configure debug-VC-WIN64A --prefix=%CD% no-asm
4) ms\do_ms
5) remove all references to unicows.lib from ms\nt.mak
7) nmake -f ms\nt.mak all
8) perl Configure VC-WIN64A --prefix=%CD% no-asm
9) ms\do_ms
A) remove all references to unicows.lib from ms\nt.mak
C) nmake -f ms\nt.mak all



Libssh2 32-bit (not yet used)
--------------

1) Edit win32\config.mk
2) Change OPENSSLINC=..\openssl\inc32
3) Change OPENSSLLIB=..\openssl\out32
4) Change the path in ZLIBINC to ..\zlib
5) Change the path in ZLIBLIB to ..\zlib
6) In the LIBS line, add the following libraries: gdi32.lib advapi32.lib
7) nmake /f NMakefile


Curl 32-bit
-----------

1) Go to your supp directory
2) rmdir deps /s /q 2> nul
3) mkdir deps\include\openssl
3) mkdir deps\lib
4) xcopy openssl\inc32\*.* deps\include /e
5) xcopy openssl\out32\*.lib deps\lib
6) cd curl\winbuild
7) nmake /f makefile.vc mode=static VC=10 WITH_SSL=static DEBUG=no ENABLE_IDN=no ENABLE_WINSSL=no
8) nmake /f makefile.vc mode=static VC=10 WITH_SSL=static DEBUG=yes ENABLE_IDN=no ENABLE_WINSSL=no


Curl 64-bit
-----------

1) Go to your supp directory
2) rmdir deps /s /q 2> nul
3) mkdir deps\include\openssl
3) mkdir deps\lib
4) xcopy openssl\inc32\*.* deps\include /e
5) xcopy openssl\out32\*.lib deps\lib
6) cd curl\winbuild
7) open MakefileBuild.vc and change the value of MACHINE to X64
7) nmake /f makefile.vc mode=static VC=10 WITH_SSL=static DEBUG=no ENABLE_IDN=no ENABLE_WINSSL=no
8) nmake /f makefile.vc mode=static VC=10 WITH_SSL=static DEBUG=yes ENABLE_IDN=no ENABLE_WINSSL=no
