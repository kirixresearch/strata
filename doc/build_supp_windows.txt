

Openssl 32-bit
--------------

1) Download Activestate Perl
2) Open Visual Studio Command Prompt and go to the openssl dir
3) perl Configure debug-VC-WIN32 --prefix=%CD% no-asm
4) ms\do_ms
5) nmake -f ms\nt.mak clean all
6) perl Configure VC-WIN32 --prefix=%CD% no-asm
7) ms\do_ms
8) nmake -f ms\nt.mak clean all


Curl 32-bit
-----------

1) Go to your supp directory
2) rmdir deps /s /q 2> nul
3) mkdir deps\include\openssl
3) mkdir deps\lib
4) xcopy openssl\inc32\*.* deps\include /e
5) xcopy openssl\out32\*.lib deps\lib
6) cd curl\winbuild
7) nmake /f makefile.vc mode=static VC=10 WITH_SSL=static DEBUG=no USE_IDN=no
8) nmake /f makefile.vc mode=static VC=10 WITH_SSL=static DEBUG=yes USE_IDN=no



Openssl 64-bit
--------------

1) Download Activestate Perl
2) Open Visual Studio Command Prompt and go to the openssl dir
3) perl Configure debug-VC-WIN64A --prefix=%CD% no-asm
4) ms\do_ms
5) remove all references to unicows.lib from ms\nt.mak
6) nmake -f ms\nt.mak clean
7) nmake -f ms\nt.mak all
8) perl Configure VC-WIN64A --prefix=%CD% no-asm
9) ms\do_ms
A) remove all references to unicows.lib from ms\nt.mak
B) nmake -f ms\nt.mak clean
C) nmake -f ms\nt.mak all


Curl 64-bit
-----------

1) Go to your supp directory
2) rmdir deps /s /q 2> nul
3) mkdir deps\include\openssl
3) mkdir deps\lib
4) xcopy openssl\inc32\*.* deps\include /e
5) xcopy openssl\out32\*.lib deps\lib
6) cd curl\winbuild
7) open MakefileBuild.vc and change the value of MACHINE to X64
8) nmake /f makefile.vc mode=static VC=10 WITH_SSL=static DEBUG=no USE_IDN=no
9) nmake /f makefile.vc mode=static VC=10 WITH_SSL=static DEBUG=yes USE_IDN=no
